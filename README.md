# 📊 Matplotlib for Data Analysis

A structured repository to learn **Matplotlib** from scratch with hands-on examples for Data Analysis. This repository covers everything from basic plotting to professional visualizations used in analytics and dashboards.

---

## 📚 Learning Roadmap

- [x] Day 1 - Introduction to Matplotlib & Line Plots
- [ ] Day 2 - Bar Charts & Horizontal Bar Charts
- [ ] Day 3 - Histograms & Distribution Visualization
- [ ] Day 4 - Scatter Plots & Correlation Analysis
- [ ] Day 5 - Pie Charts & Custom Styling
- [ ] Day 6 - Multiple Plots (Subplots)
- [ ] Day 7 - Figure Customization (Titles, Labels, Legends, Grid)
- [ ] Day 8 - Colors, Markers & Line Styles
- [ ] Day 9 - Saving Figures & High-Quality Images
- [ ] Day 10 - Real Dataset Visualization Project

---

## 🛠️ Tech Stack

- Python
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook / VS Code

---

## 🎯 Learning Outcomes

After completing this repository, we will be able to:

- Create professional data visualizations
- Analyze trends using line charts
- Compare categories using bar charts
- Visualize distributions using histograms
- Explore relationships using scatter plots
- Build multiple charts using subplots
- Customize plots with colors, legends, grids, and styles
- Save publication-quality figures
- Create visualizations for Data Analyst portfolio projects
